# [GAN 인 액션]

Gan 인 액션 책을 통해 적대적 생성 신경망 공부하기

GAN 인 액션(한빛미디어, 2020)의 실습 코드를 저장 해 두었습니다.

모든 예제 코드는 주피터 노트북으로 제공되며 **구글 코랩(Colab)** 에서 실행할 수 있도록 각 주피터노트북에 링크를 달아뒀습니다.

이 코드는 텐서플로 2.1 버전, __Tesla V100 32gb__ 에서 테스트했습니다.
